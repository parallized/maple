# Homepage Image Guide

本项目的图片素材采用“缺失回退”策略：如果图片文件不存在，页面会自动展示占位块；你只需要按约定放入文件即可生效，无需改代码。

## 放置目录

把图片放到：`apps/homepage/public/images/`

## 需要补齐的图片

1) `hero.png`
   - 用途：首屏主视觉（产品界面截图 / 宣传图）
   - 建议尺寸：`1200 x 1500`（或更高，竖向更适配首屏布局）
   - 内容建议：展示 Maple 的看板 + 执行日志/状态回写的组合画面

2) `screenshot-board.png`
   - 用途：功能图 1（看板 / 任务执行）
   - 建议尺寸：`1600 x 900`（16:9）
   - 内容建议：Board 视图，能看到状态列与任务卡片层级

3) `screenshot-settings.png`
   - 用途：功能图 2（安装 / Worker 配置）
   - 建议尺寸：`1600 x 900`（16:9）
   - 内容建议：Settings 视图，突出 Worker/MCP 安装与探测状态

## 提示

- 文件名必须完全一致（大小写也要一致）。
- 图片建议导出为 PNG，避免过度压缩导致文字发糊。
